Easy Batch Tutorials

1. Introduction

Easy Batch tutorials are organized into 3 categories:

  • Basic tutorials : These are a set of Hello World samples which will allow you to understand the basic APIs of Easy Batch and how to use them effectively. These tutorials show how to use Easy Batch main features such as record reading, filtering and mapping on a variety of data source types (Flat files, XML file, database, etc).
  • Intermediate Tutorials : These tutorials are more elaborate than Hello World samples and show how to extend Easy Batch built-in components on more complex data models.
  • Advanced Tutorials : show the usage of advanced features of Easy Batch such as processing data in parallel, scheduling Easy Batch instances, Monitoring Easy Batch execution at runtime, etc.
