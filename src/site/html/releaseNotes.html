Release notes

Version 1.0.2 (2012-12-01)

This is the second maintenance release in the 1.0.x branch. This release adds some minor changes to support loading configuration parameters file from the classpath. It also fixes a performance issue due to String concatenation. For all details about changes in this release, please refer to the change log.

Version 1.0.1 (2012-11-01)

This is the first maintenance release in the 1.0.x branch. The release includes multiple performance optimizations, bugfixes and documentation updates. The full list of changes may be found in change log.

Version 1.0.0 (2012-10-14)

This is the first CB4J release. The main features of this release are :

  • Initial core API and Implementation
  • Common data type validators

Roadmap

Many features are currently under development and will be available in next CB4J versions:

  • version 1.1:
    1. Add configuration parameter to support data qualifier character
    2. Add configuration parameter to support whitespaces trimming
    3. Add configuration parameter to support execution abort on first reject
    4. Add JMX monitoring support
    5. Add a log file for records processed with error
  • version 1.2 :
    1. Add Fixed-Length Record parsing support
    2. Add annotation-based configuration support
    3. Add xml-based configuration support
  • version 1.3 :
    1. Add CB4J-Spring integration module
    2. Add statistics module to report processing time for each record
    3. Add transaction support
  • version 2.0 :
    1. Add GUI module that will enable graphical configuration, execution and monitoring of CB4J
    2. Add parallelism support to split input file into multiple partitions, run parallel CB4J instances and then merge results
