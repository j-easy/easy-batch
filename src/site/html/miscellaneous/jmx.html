Monitoring CB4J with JMX

Description

CB4J allows you to monitor batch execution and progress using JMX. By default, CB4J enables at startup a JMX MBean named net.benas.cb4j.core.jmx.BatchMonitorMBean.

You can use any standard JMX compliant tool to monitor CB4J's JMX MBean attributes such as:

  • Batch execution status (initializing, running or finalizing)
  • Execution progress
  • The number of ignored, rejected and processed records
  • etc
